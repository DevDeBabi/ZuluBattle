Politique de Confidentialité

Dernière mise à jour : 28/10/2024

Bienvenue dans notre jeu ! Nous attachons une grande importance à la confidentialité de nos joueurs et nous nous engageons à protéger les informations personnelles que nous collectons. Cette politique de confidentialité explique les types de données que nous recueillons, la manière dont nous les utilisons, et les droits des utilisateurs concernant leurs informations.

1. Types de Données Collectées

Lorsque vous utilisez notre jeu, nous pouvons collecter les données suivantes :

  • Données personnelles :
    • Pseudo
    • Adresse e-mail
    • Emplacement géographique
  • Données d'utilisation :
    • Score
    • Interactions (actuellement en développement)

2. Méthodes de Collecte de Données

Les données sont collectées directement auprès des utilisateurs lorsqu'ils s'inscrivent ou interagissent dans le jeu. Des données d'utilisation peuvent également être collectées automatiquement pour améliorer l'expérience de jeu.

3. Utilisation des Données

Les données collectées sont utilisées dans les buts suivants :

  • Analyse des données : Pour comprendre les habitudes des joueurs et améliorer le jeu.
  • Gestion du classement : Afin de classer les joueurs en fonction de leurs performances.
  • Envoi de notifications : Pour informer les utilisateurs des mises à jour importantes et événements en jeu.

4. Stockage et Sécurité des Données

Les données sont stockées sur les serveurs de Unity, et certaines informations peuvent également être stockées sur Firebase. Nous utilisons des mesures de sécurité pour protéger les données personnelles contre tout accès non autorisé ou toute utilisation abusive.

5. Droits des Utilisateurs

Les utilisateurs ont les droits suivants concernant leurs informations personnelles :

  • Accès et modifications : Les utilisateurs peuvent modifier leur pseudo directement dans le jeu.
  • Suppression des données : Pour supprimer toutes leurs informations personnelles, les utilisateurs peuvent nous contacter à l'adresse game2babi.g2b@gmail.com.

6. Partage des Données

Nous ne partageons pas les données personnelles avec des tiers, sauf si cela est nécessaire pour fournir les services du jeu ou respecter une obligation légale.

7. Âge Minimum des Utilisateurs

Le jeu est destiné à un public de 12 ans et plus. Nous ne collectons pas intentionnellement de données auprès d’enfants de moins de 12 ans sans le consentement parental approprié.

8. Contact

Pour toute question ou demande concernant notre politique de confidentialité, vous pouvez nous contacter à l'adresse suivante :
game2babi.g2b@gmail.com

Cette politique de confidentialité peut être modifiée à tout moment, alors nous vous invitons à la consulter régulièrement pour rester informé.
